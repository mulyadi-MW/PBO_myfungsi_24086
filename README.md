# PBO_myfungsi_24086
PROJEK PBO UNTUK MEMENUHI MATA KULIAH PBO
Program ini dibuat menggunakan bahasa pemrograman Python dengan menerapkan konsep Object-Oriented Programming (OOP), khususnya pewarisan (inheritance) dan penggunaan metode super(). Program ini bertujuan untuk memodelkan sebuah kendaraan, khususnya motor, beserta informasi dan aktivitasnya.
Program terdiri dari dua kelas utama, yaitu kelas Kendaraan sebagai kelas induk dan kelas Motor sebagai kelas turunan.
Kelas Kendaraan berfungsi sebagai kelas dasar yang menyimpan atribut umum berupa merk kendaraan. Kelas ini memiliki sebuah constructor (__init__) yang digunakan untuk menginisialisasi nilai merk serta sebuah method info() yang berfungsi untuk menampilkan informasi merk kendaraan.
Kelas Motor merupakan turunan dari kelas Kendaraan yang mewarisi seluruh atribut dan method dari kelas induk. Selain atribut yang diwarisi, kelas Motor memiliki atribut tambahan yaitu warna motor dan jarak tempuh. Pada saat objek motor dibuat, constructor kelas Motor memanggil constructor kelas induk menggunakan fungsi super() untuk menginisialisasi atribut merk, kemudian menambahkan atribut warna dan jarak tempuh awal motor.
Kelas Motor juga melakukan override terhadap method info() milik kelas induk. Method ini tetap menampilkan informasi merk kendaraan dari kelas induk, kemudian menambahkan informasi khusus motor berupa warna dan jarak tempuh. Selain itu, kelas Motor memiliki method jalan() yang digunakan untuk mensimulasikan aktivitas motor saat berjalan, serta method tambah_jarak() yang berfungsi untuk menambahkan jarak tempuh motor berdasarkan jarak yang ditempuh.
Pada bagian akhir program, dibuat sebuah objek dari kelas Motor dengan merk dan warna tertentu. Program kemudian menampilkan informasi motor, menjalankan fungsi berjalan, menambahkan jarak tempuh, dan menampilkan kembali informasi motor setelah jarak tempuh bertambah. Dengan demikian, program ini dapat digunakan untuk memahami konsep dasar OOP dalam Python secara sederhana dan terstruktur
